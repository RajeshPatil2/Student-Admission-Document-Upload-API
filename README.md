
STUDENT ADMISSION DOCUMENT UPLOAD API
==============================

**Project Type**: Backend REST API  
**Technology**: Java, Spring Boot, Spring Data JPA, Hibernate  
**Database**: MySQL  
**Build Tool**: Maven  
**Server Port**: 1999  

--------------------------------------------------
PROJECT OVERVIEW
--------------------------------------------------
This project is a **Student Admission Management System** built using **Spring Boot**.

The main purpose of this project is to:
- Store **student personal details**
- Upload and save **admission documents**
- Upload and save **semester result documents**
- Maintain proper **One-to-One relationships** between entities

This project demonstrates **real-world backend development concepts** used in production systems.

--------------------------------------------------
KEY FEATURES
--------------------------------------------------
- **Student Registration API**
- **Multipart File Upload** (Marksheet, Aadhaar, PAN, Result files)
- **One-to-One Entity Mapping**
- **RESTful API Design**
- **JSON + File Handling**
- **Database Persistence using JPA**
- **Clean Layered Architecture**
